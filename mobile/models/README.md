# models/

Здесь будут храниться типы и интерфейсы доменных сущностей приложения GoNext:

- Place (место);
- Trip (поездка);
- TripPlace (место в поездке);
- Highlight (достопримечательность / событие).

Подробное описание и реализация моделей запланированы в `PLAN.md`, раздел
«2.3. Крупные этапы разработки (MVP) → Этап 2. Локальная база данных и модели».
