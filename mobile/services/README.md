# services/

В этой директории планируется разместить прикладные сервисы, в том числе:

- слой доступа к локальной базе данных (SQLite) для сущностей Place, Trip, TripPlace, Highlight;
- вспомогательные сервисы (например, работа с геолокацией или файловой системой для фотографий).

Создание и реализация конкретных сервисов запланированы в `PLAN.md`,
раздел «2.3. Крупные этапы разработки (MVP) → Этап 2. Локальная база данных и модели».
