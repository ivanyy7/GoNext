## PLAN.md — исходный план проекта GoNext

Этот файл фиксирует исходный (стартовый) план проекта.  
В дальнейшем по правилам проекта мы **не переписываем** уже написанное, а только дополняем план ниже по мере развития.

### 1. Текущее состояние

- Проект создан, начальная структура и правила Cursor подключены.
- Исходный функциональный план ещё **не сформулирован** — будет добавлен после постановки первой задачи.

### 2. Черновой каркас плана (заполнится позже)

Ниже будут появляться разделы, когда ты озвучишь первую задачу / промт:

- Общие цели проекта.
- Основные пользовательские сценарии.
- Крупные этапы разработки (MVP, улучшения, интеграции и т.п.).

### 3. Изменения плана (будущее расширение)

Когда фактическое развитие проекта начнёт расходиться с исходным планом, мы будем:

- не изменять старые пункты выше,
- а **добавлять** сюда новые блоки с пометками «Актуализированная версия плана» или «Изменения в плане».
